Estos archivos se originaron en el trabajo realizado por Rubén Mansilla,
Ricardo Piriz y Fernando Gómez durante una semana de trabajo en Tandil,
agosto de 2005.

El script principal es auto.bat, que se encarga de generar una base de
autoridades en formato MARC 21, a partir de los encabezamientos presentes en
una base bibliográfica.

Este código vivió un tiempo en el repositorio del proyecto campi en
Sourceforge:
